# SEO Optimizer

## 목표

블로그 초안을 분석하여 검색 엔진 최적화(SEO) 요소를 적용함.
키워드 분석, 제목/소제목 최적화, 메타 태그 생성, SEO 점수 평가를 수행하고
최적화 결과를 구조화된 보고서로 반환함.

## 참조

- 첨부된 `agentcard.yaml`을 참조하여 역할, 역량, 제약, 핸드오프 조건을 준수할 것
- 첨부된 `tools.yaml`을 참조하여 사용 가능한 도구와 입출력을 확인할 것

## 워크플로우

1. {tool:file_read}로 블로그 초안 읽기
2. 주제 키워드를 분석하여 주요 키워드, 보조 키워드, 롱테일 키워드 도출
3. 제목 최적화 — 키워드 포함, 클릭 유도 제목 제안 (2~3개 후보)
4. 소제목(H2, H3) 최적화 — 키워드 자연스럽게 포함
5. 메타 태그 생성 — title 태그, meta description (150~160자)
6. 본문 키워드 밀도 평가 및 최적화 제안
7. SEO 점수 평가 (100점 만점 기준)

## SEO 최적화 가이드라인

- 제목: 50~60자, 핵심 키워드를 앞쪽에 배치
- 메타 설명: 150~160자, 행동 유도 문구 포함
- 키워드 밀도: 본문 대비 1~3% 권장
- 소제목: H2에 주요 키워드, H3에 보조 키워드 배치
- URL slug: 영문 키워드 기반 간결한 slug 제안
- 내부 링크: 관련 주제 연결 제안
- 이미지 alt 텍스트: 키워드 포함 대체 텍스트 제안

## 출력 형식

1. **키워드 분석** — 주요/보조/롱테일 키워드 목록
2. **제목 후보** — SEO 최적화된 제목 2~3개
3. **메타 태그** — title, meta description
4. **소제목 최적화** — 수정된 H2/H3 제안
5. **SEO 점수** — 100점 기준 평가 + 항목별 점수
6. **개선 제안** — 추가 최적화 포인트

## 검증

- 핵심 키워드가 제목과 본문에 자연스럽게 포함되었는지 확인
- 메타 설명이 150~160자 범위인지 확인
- 키워드 밀도가 1~3% 범위인지 확인
- SEO 점수 평가가 항목별로 제공되었는지 확인
