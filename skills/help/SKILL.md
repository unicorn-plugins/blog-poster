# Help

[BLOG-POSTER HELP 활성화]

---

## 목표

blog-poster 플러그인의 사용 가능한 명령, 자동 라우팅 규칙, 주요 기능을 안내함.
런타임 상주 파일(CLAUDE.md)에 라우팅 테이블을 등록하는 대신,
이 스킬이 호출 시에만 토큰을 사용하여 사용자 발견성을 제공함.

## 워크플로우

**중요: 추가적인 파일 탐색이나 에이전트 위임 없이, 아래 내용을 즉시 사용자에게 출력하세요.**

### 사용 가능한 명령

| 명령 | 설명 |
|------|------|
| `/blog-poster:setup` | 플러그인 초기 설정 (MCP 서버, 도구, 의존성 설치) |
| `/blog-poster:help` | 사용 안내 (이 화면) |
| `/blog-poster:write-post` | 블로그 글 작성 메인 워크플로우 |

### 자동 라우팅

슬래시 명령 없이 자연어로도 사용할 수 있습니다:

| 키워드/표현 | 라우팅 스킬 |
|------------|-----------|
| "블로그 써줘", "글 작성해줘", "포스트 만들어" | write-post |
| "도움말", "뭘 할 수 있어" | help |
| "설정", "셋업" | setup |

### 워크플로우 안내

`/blog-poster:write-post` 실행 시 6단계로 진행됩니다:

1. **주제 입력** — 블로그 주제(키워드)를 입력
2. **리서치** — 주제 관련 웹 검색, 기술 문서 조사
3. **초안 작성** — 리서치 기반 블로그 초안 작성
4. **SEO 최적화** — 키워드 분석, 메타 태그, 제목 최적화
5. **이미지 생성** — AI로 블로그 삽화/썸네일 생성
6. **최종 문서** — Word(.docx) 파일로 출력

### 에이전트 구성

| 에이전트 | 역할 |
|----------|------|
| researcher | 웹 검색, 기술 문서 조사, 트렌드 분석 |
| writer | 블로그 본문 작성, Word 문서 생성 |
| seo-optimizer | 키워드 분석, 메타 태그, SEO 점수 평가 |
| image-creator | AI 이미지 생성 (Gemini 기반) |

### 요구사항

| 항목 | 필수 여부 | 용도 |
|------|----------|------|
| context7 MCP | 선택 | 기술 문서 검색 (리서치 품질 향상) |
| GEMINI_API_KEY | 선택 | AI 이미지 생성 (없으면 이미지 단계 건너뜀) |
| python-docx | 필수 | Word 문서 출력 |

## MUST 규칙

| # | 규칙 |
|---|------|
| 1 | 추가 파일 탐색 없이 즉시 안내 내용을 출력 |
| 2 | 에이전트 위임 없이 직결형으로 실행 |
| 3 | 최신 명령어와 워크플로우 정보를 정확히 안내 |

|---|----------|

| 2 | 파일을 생성하거나 수정하지 않음 |

## 검증 체크리스트

- [ ] 사용 가능한 명령이 모두 나열되어 있는가
- [ ] 자동 라우팅 규칙이 core 스킬과 일치하는가
- [ ] 워크플로우 6단계가 정확히 기술되어 있는가
- [ ] 에이전트 위임 없이 즉시 출력하는가
- [ ] 요구사항(MCP, API Key, 의존성)이 안내되어 있는가
