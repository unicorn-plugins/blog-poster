## 핵심 인사이트

- **세대적 도약**: SWE-bench 93.9%, Terminal-Bench 92.1%, USAMO 97.6% — 단순 점수 향상이 아닌  
  기존 최고 모델 대비 13%p 이상의 격차로 '새로운 모델 클래스' 수준
- **의도적 제한 배포**: 공격적 사이버보안 능력이 너무 강력하여 일반 공개 불가 판단,  
  Project Glasswing이라는 방어 목적 이니셔티브로 통제된 배포 선택
- **개발자 접근 불가 현실**: 공개 API 없음, 셀프 서비스 없음, 가격 미정 — 일반 개발자는  
  Opus 4.7(87.6%)이 현실적 대안
- **사이버보안 패러다임 전환**: AI가 제로데이 취약점을 자동 탐지·익스플로잇하는 시대  
  진입, 방어자/공격자 모두에게 게임 체인저
- **보안 사고**: 출시 당일 비인가 그룹이 제3자 벤더 환경을 통해 접근 성공 — AI  
  역량이 강력할수록 공급망 보안의 중요성 부각
- **정렬(Alignment) 유지**: 고성능에도 불구하고 Anthropic 자체 평가에서 가장  
  잘 정렬된 모델로 분류
