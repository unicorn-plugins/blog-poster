# Claude Mythos: 새로운 모델 클래스의 등장

## 개요

Anthropic이 2026년 4월 8일, 코드네임 'Capybara'로 알려진 **Claude Mythos Preview**를 공개함.  
단순 업그레이드가 아닌 새로운 모델 클래스(model class)로, 코딩·추론·사이버보안 세 영역에서  
동시에 최고 성능을 기록함. 주요 OS와 브라우저에서 수천 건의 제로데이 취약점을 자동 탐지하며,  
최장 27년 전 OpenBSD 취약점까지 발견함.

## 벤치마크: 세대적 도약

- **SWE-bench Verified**: 93.9% — GitHub 실제 이슈를 20회 중 19회 해결  
- **Opus 4.6(80.8%) 대비** +13.1%p, GPT-5.3 Codex(85%) 대비 +8.9%p  
- **Terminal-Bench 2.1** (4시간 타임아웃): 92.1%  
- **USAMO(수학 올림피아드)**: 97.6%  

2024년 상위 모델 기준(40~55%)과 비교하면 점진적 개선이 아닌 세대적 도약임.  
사이버보안 실험에서 Opus 4.6이 Firefox 147 JS 엔진 취약점을 2회 익스플로잇한 반면,  
Mythos Preview는 동일 조건에서 181회 성공함.

## 제한 배포: Project Glasswing

공격적 능력이 강력한 만큼 Anthropic은 일반 공개 대신 통제된 배포를 선택함.  
**Project Glasswing**은 방어적 사이버보안 목적으로만 Mythos를 제공하는 이니셔티브로,  
Apple·Amazon·Microsoft·Google 등 40개 이상 기업에 1억 달러 이상의 크레딧을 제공함.  
활용 범위는 취약점 탐지·침투 테스트·엔드포인트 보안에 한정됨.

## 개발자 접근 현실

현시점에서 공개 API와 셀프 서비스 가입은 불가함. 모델 ID  
`claude-mythos-preview-20260408`은 초청 기반 엔터프라이즈 고객에게만 열려 있음.  
일반 개발자의 현실적 대안은 **Claude Opus 4.7(SWE-bench 87.6%)** 임.  
출시 직후 비인가 그룹이 제3자 벤더 API 키를 통해 접근에 성공한 사고는,  
고성능 모델일수록 공급망 보안 관리가 핵심 과제임을 실증함.
