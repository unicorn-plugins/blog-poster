## 핵심 인사이트  
- Claude Mythos는 일반 공개 LLM이 아니라 방어적 사이버보안 연구 프리뷰 성격임  
- 핵심 차별점은 채팅 품질보다 코드 이해, 취약점 탐지, 익스플로잇 구성, 장시간 에이전트 수행 능력임  
- 성능 수치는 매우 인상적이지만, 일부는 내부 구현·내부 조건 평가이므로 공개 리더보드 절대값처럼 쓰면 과장 위험 존재  
- Anthropic의 현재 전략은 Mythos 직접 공개보다 Project Glasswing로 제한 운영하고, Opus 4.7에서 안전장치를 실전 검증하는 흐름임  
- 개발자 관점에서 지금 중요한 질문은 "언제 써보나"보다 "AI가 취약점 탐지와 패치 속도를 어떻게 바꾸나"임
